# Tableau Viz Extensions

A collection of custom D3-powered Tableau Viz Extensions for loyalty and marketing analytics.

[View Gallery](gallery.html)

## Extensions

### High Priority (7)

- Cohort Retention Heatmap
- Funnel Chart
- Sankey Diagram
- RFM Scatter
- Treemap
- Waterfall Chart
- Survival/Churn Curve

### Medium Priority (7)

- Bubble Chart
- Network Graph
- Gauge Chart
- Histogram
- Stacked Area Chart
- CLV vs Churn Quadrant
- Lollipop Chart

## Local Dev

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080` in your browser.
