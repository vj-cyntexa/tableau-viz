Tableau Viz Extensions Cyntexa

Custom D3.js Viz Extensions for Tableau — loyalty analytics, CRM, and retail dashboards

Live Extensions
Multi-Line Chart
Time series with multiple configurable lines per dimension segment. Supports custom colors, line thickness, and crosshair tooltip.
Use for: trend analysis by loyalty tier, revenue over time per segment
Test Page Extension
Bar Chart
Stacked, grouped, horizontal, and vertical bar chart with optional dotted average reference line. Fully configurable layout and orientation.
Use for: segment comparison, store performance ranking, revenue by channel
Test Page Extension
Venn Diagram
Multi-panel Venn diagram for set overlap analysis. Supports up to 3+ overlapping sets per panel, multi-group tiling, per-circle color controls, and Diagram / List view toggle.
Use for: loyalty segment overlap, customer membership across programmes
Test Page Spaced Variation Extension
KPI Card
Single-metric display card with four modes: Value, Comparison, Delta, and Sparkline. Configurable via mode toggle bar.
Use for: executive dashboards, headline metrics, period-over-period comparison
Test Page Extension
Donut Chart
Three donut chart modes: Standard, Exploded (offset slices), and Multi-Ring (concentric rings). Smooth arc transitions.
Use for: tier composition, channel mix, category share
Test Page Extension
Bar-Line Combo
Dual-axis chart with bars on the left axis and a line overlay on the right axis. Supports optional color grouping for stacked bars.
Use for: revenue (bars) vs. customer count (line), sales vs. redemption rate
Test Page Extension
Relationship Table
Heatmap matrix for any two categorical dimensions. Color intensity = value. Sortable columns, luminance-adaptive text, hover tooltips.
Use for: channel × transaction type analysis, region × product heatmap
Test Page Extension
Affinity Matrix
Product co-purchase matrix with Count, Lift, and Support metric toggle. Diverging color scale for Lift (purple = avoidance, orange = affinity). Sortable, symmetric mode, diagonal highlighting.
Use for: product affinity, market basket analysis, cross-sell opportunity identification
