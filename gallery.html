Tableau Viz Extensions Cyntexa

Custom D3.js Viz Extensions for Tableau — loyalty analytics, CRM, and retail dashboards

Live Extensions
Multi-Line Chart
Time series with multiple configurable lines per dimension segment. Supports custom colors, line thickness, and crosshair tooltip.
Use for: trend analysis by loyalty tier, revenue over time per segment
Test Page Extension
Bar Chart
Stacked, grouped, horizontal, and vertical bar chart with optional dotted average reference line. Fully configurable layout and orientation.
Use for: segment comparison, store performance ranking, revenue by channel
Test Page Extension
Venn Diagram
Multi-panel Venn diagram for set overlap analysis. Supports up to 3+ overlapping sets per panel, multi-group tiling, per-circle color controls, and Diagram / List view toggle.
Use for: loyalty segment overlap, customer membership across programmes
Test Page Extension
KPI Card
Single-metric display card with four modes: Value, Comparison, Delta, and Sparkline. Configurable via mode toggle bar.
Use for: executive dashboards, headline metrics, period-over-period comparison
Test Page Extension
Donut Chart
Three donut chart modes: Standard, Exploded (offset slices), and Multi-Ring (concentric rings). Smooth arc transitions.
Use for: tier composition, channel mix, category share
Test Page Extension
Bar-Line Combo
Dual-axis chart with bars on the left axis and a line overlay on the right axis. Supports optional color grouping for stacked bars.
Use for: revenue (bars) vs. customer count (line), sales vs. redemption rate
Test Page Extension
Relationship Table
Heatmap matrix for any two categorical dimensions. Color intensity = value. Sortable columns, luminance-adaptive text, hover tooltips.
Use for: channel × transaction type analysis, region × product heatmap
Test Page Extension
Affinity Matrix
Product co-purchase matrix with Count, Lift, and Support metric toggle. Diverging color scale for Lift (purple = avoidance, orange = affinity). Sortable, symmetric mode, diagonal highlighting.
Use for: product affinity, market basket analysis, cross-sell opportunity identification
Test Page Extension
Planned Extensions In Progress
Cohort Retention Heatmap
Grid of acquisition cohorts × periods. Cell color = retention rate %. The standard chart for loyalty retention diagnostics.
Use for: cohort retention, acquisition quality comparison
Funnel Chart
Horizontal or vertical funnel with trapezoid stages showing volume and conversion rate at each step.
Use for: enrollment funnel, redemption flow, onboarding drop-off
Sankey / Flow Diagram
Node-link flow diagram showing customer migration between loyalty tiers or touchpoints. Edge width = flow volume.
Use for: tier migration, customer journey, segment flow
RFM Scatter Plot
Recency × Frequency scatter with Monetary as bubble size. Quadrant lines at median values create the classic RFM view.
Use for: RFM segmentation, customer prioritization
Treemap
Nested rectangle chart where area = size measure. 1-level or 2-level hierarchy. Color = category or second measure.
Use for: revenue by segment hierarchy, tier × category breakdown
Waterfall Chart
Sequential bars showing running total with positive/negative increments. Connector lines between bars.
Use for: net member movement, revenue attribution, P&L bridge
Survival / Churn Curve
Kaplan-Meier style survival curve per segment. Shows probability of remaining active at each time period. Optional confidence interval bands.
Use for: churn analysis, retention curve by acquisition cohort
Bubble Chart
3-measure scatter plot: X, Y, and bubble size. Quadrant lines at medians. Segment color coding.
Use for: segment prioritization, CLV × frequency × count
Network / Force Graph
Force-directed product co-purchase network. Nodes = products, edges = lift score. Clusters emerge for naturally co-purchased products.
Use for: product affinity clusters, basket analysis visualization
Gauge / Progress Arc
Semi-circular arc gauge showing KPI value vs target. Traffic-light color thresholds. Configurable arc thickness.
Use for: redemption rate vs goal, tier fill rate, program KPIs
Histogram / Distribution
Frequency distribution with configurable bin count, optional KDE overlay, and per-segment overlapping mode.
Use for: CLV distribution, order value spread, points balance histogram
Stacked Area Chart
Time series with stacked/normalized/grouped line modes. Shows tier mix evolution over time.
Use for: tier composition over time, revenue mix trend
CLV vs Churn Risk Quadrant
Scatter plot with configurable quadrant lines at CLV and churn risk thresholds. Labels: Protect & Grow, Urgent Retention, Nurture, Deprioritize.
Use for: retention spend prioritization, segment action matrix
Lollipop / Ranking Chart
Clean ranked lollipop chart (stem + dot) with benchmark line. Horizontal or vertical orientation.
Use for: store rankings, campaign leaderboard, segment KPI comparison
